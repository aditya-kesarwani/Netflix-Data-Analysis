# Netflix-Data-Analysis

This project analyzes Netflix movies and TV show data using Python, Pandas, NumPy, and Matplotlib.

## Features

- Handles missing values in dataset
- Cleans Netflix dataset
- Calculates average ratings
- Creates data visualization charts
- Saves cleaned dataset into new CSV file

## Technologies Used

- Python
- Pandas
- Matplotlib

## Files Included

- netflix_dataset.csv
- netflix_data_cleaning.py
- cleaned_netflix_dataset.csv
- netflix_visualization.py
- netflix_graph.png

## Project Workflow

1. Read Netflix dataset
2. Check missing values
3. Fill missing values using average
4. Create cleaned dataset
5. Generate visualization graph

## Output

The project generates:
- Cleaned Netflix dataset
- Netflix ratings visualization graph
